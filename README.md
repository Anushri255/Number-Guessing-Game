# Number Guessing Game

A simple web-based Number Guessing Game built with HTML, CSS, and JavaScript. The objective is to guess a randomly generated number within a specified range. The game provides feedback on whether the guess is too high, too low, or correct, and keeps track of the number of attempts.

## Features

- **Guess a number**: Input your guess and check if it's correct.
- **Feedback**: Receive feedback if the guess is too high or too low.
- **Attempts Counter**: Track how many tries it took to guess the correct number.
- **Reset Game**: Restart the game to guess a new number.

## Getting Started

To run the Number Guessing Game locally:

1. **Clone the Repository**:
    ```bash
    git clone <repository-url>
    ```

2. **Navigate to the Project Directory**:
    ```bash
    cd <project-directory>
    ```

3. **Open `number-guessing.html`**:
   
    Open `number-guessing.html` in your web browser to start playing the game.

## How It Works

1. **Generate Random Number**: A random number is generated between 1 and 100 (or another range if you modify the code).
2. **Submit Guess**: Enter a guess in the input field and submit.
3. **Receive Feedback**: The game will tell you if your guess is too low, too high, or correct.
4. **Track Attempts**: The number of attempts is displayed and updated as you guess.
5. **Reset Game**: Click the "Reset Game" button to start a new game with a new random number.
